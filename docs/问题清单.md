## 新检查单（等待测试）

### 核心导出

- [ ] 创建 1 个 Start 节点并连接 If/Log，导出 `.code`，文件非空且包含预期内容
- [ ] 创建 2 个 Start 节点分别连接不同分支，导出 `.code` 生成 2 个 label
- [ ] 空图导出 `.code` 应提示/生成空文件（行为可接受）

### 节点参数编辑

- [ ] 选中 Start 节点，右侧面板显示 `label` 参数
- [ ] 将 `label` 从空值改为字符串，JSON 预览同步更新
- [ ] 选中 If 节点，修改 `condition` 参数（布尔/字符串/数字），JSON 预览同步更新
- [ ] 修改参数后撤销/重做，参数恢复

### 端口与折叠

- [ ] 节点上显示端口圆点和端口标签（如 In/Out）
- [ ] 右键节点 → 折叠/展开，参数预览隐藏/显示，端口标签始终可见
- [ ] 折叠后节点高度应缩小

### 复制/粘贴

- [ ] Ctrl+C 复制选中节点，状态栏提示“已复制”
- [ ] Ctrl+V 在鼠标位置粘贴，新节点被选中且置顶
- [ ] 粘贴后新旧节点重叠时，拖动新节点（而非旧节点）
- [ ] 右键空白处 → 粘贴，功能正常
- [ ] 剪贴板为空时，右键空白处“粘贴”按钮置灰
- [ ] 搜索窗口打开时，Ctrl+C/V 不触发节点复制/粘贴（保留搜索框文本编辑）

### 性能

- [ ] 添加 100 个 Start 节点，帧率≥30 fps
- [ ] 缩放/平移 100 节点画布，帧率稳定

### 新建确认

- [ ] 画布有节点时点击“新建”，弹出确认对话框
- [ ] 选择“否”保留当前图，选择“是”清空
- [ ] 空画布时点击“新建”不弹窗

### 文件与 JSON

- [ ] Ctrl+S 保存首次弹出对话框，之后保存到同一文件
- [ ] 打开损坏 JSON 时，弹出错误对话框并显示状态栏错误
- [ ] 打开有效 JSON 后节点/连线/参数一致
- [ ] JSON 预览只在图变化时更新，不每帧闪烁

### 搜索窗口

- [ ] 空格键打开/关闭搜索窗口
- [ ] Escape 关闭搜索窗口
- [ ] “X 关闭”按钮正常显示且可点击关闭
