## 临时检查单（第二轮修复 - 待验证）

### 交互

- [ ] 单击节点可选中（之前仅能拖拽选中）
- [ ] 单击空白处可取消选中
- [ ] 单击连线可选中（蓝色发光高亮）
- [ ] 框选 Window 模式：左→右拖拽，蓝色实线框，仅选中完全包含的节点
- [ ] 框选 Crossing 模式：右→左拖拽，绿色虚线框，相交即选中
- [ ] Shift + 框选追加选中；Ctrl + 框选减选
- [ ] 在节点上 Shift/Ctrl + 拖拽 → 框选模式（不触发单节点拖拽）
- [ ] 框选矩形覆盖连线时，连线被选中
- [ ] 多选节点后拖拽任一节点，所有选中节点同步移动且置顶显示
- [ ] 从输出端口拖线，兼容输入端口显示高亮圆环
- [ ] 拖线悬停不兼容/已占用/成环端口，端口圆点和连线变色（红/橙），释放拒绝连接
- [ ] 空格键打开/关闭搜索窗口
- [ ] 搜索窗口可按 Escape 关闭，按 ✕ 按钮关闭
- [ ] 右键节点菜单：复制/粘贴/删除/折叠均可正常执行
- [ ] 右键空白处菜单：粘贴正常执行（需已有剪贴板内容）
- [ ] Delete 键删除选中节点/连线（包含级联删边），撤销可恢复

### 快捷键

- [ ] Ctrl+C 复制选中节点
- [ ] Ctrl+V 在鼠标位置粘贴
- [ ] Ctrl+Z 撤销
- [ ] Ctrl+Y / Ctrl+Shift+Z 重做
- [ ] Ctrl+S 弹出保存对话框 / 保存到当前文件
- [ ] Delete 删除选中内容

### 文件与数据

- [ ] 工具栏"新建"清空画布，显示欢迎提示
- [ ] 工具栏"打开 JSON"可弹出文件选择对话框
- [ ] 工具栏"保存 (Ctrl+S)"首次保存弹出对话框，之后保存到同一文件
- [ ] 工具栏"导出 JSON"弹出保存对话框
- [ ] 工具栏"导出 .code"弹出保存对话框
- [ ] 加载有效 JSON 后节点位置、连线、参数一致
- [ ] 加载损坏 JSON 不 panic，状态栏显示错误信息

### 验证与极端场景

- [ ] 连接成环（A→B→A），相关节点边框变红，状态栏显示错误数量
- [ ] 添加 100 个 Start 节点，帧率保持≥30（视口裁剪已启用）
- [ ] 保存的 JSON 中 `version` 为 `"1.3"`

### 视觉效果

- [ ] 选中连线显示蓝色发光（辉光 + SELECTED_GLOW 核心色），明显可辨识
- [ ] 连线始终在节点下层渲染，不遮挡节点
- [ ] 选中节点置顶渲染，不会被未选中节点遮挡
