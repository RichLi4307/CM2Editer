## P0 — 阻塞

### Log 节点参数编辑只能输入数字

**症状**：新建或 UI 创建的 Log 节点，属性面板 output 栏输入非数字字符串被替换为 `[]`。导入 test1.json 中的 Log 节点可正常编辑。
**推测**：可能是 `default_value()` 创建的默认值 `""` 与加载的 `"hello"` 在属性面板的解析路径上有差异，或 `param_value_to_string` 空字符串处理有问题。
**文件**：`src/ui/panels/properties.rs`

---

## P1 — 核心体验

### 空图导出无警告反馈

**症状**：空图（如 test4.json）导出 .code 成功，但文件为空，状态栏无任何警告。
**预期**：空图导出时弹窗或状态栏提示"图为空，导出内容为空"。

### If 节点 condition 仅可手动勾选

**症状**：无法通过 dataflow 连接其他节点的输出端口到 If 的 condition 输入。目前只能手动勾选 true/false。
**处理**：归入 DataFlow 重构计划一起处理。

---

## P2 — 体验问题

### 2 个 Start 节点分叉时导出为单一 main 标签

**症状**：创建 S1→A→B 和 S2→C→D 两套独立流程，导出 .code 后两个 Start 都在 `main:` 下，未生成独立标签。
**文件**：`src/code_gen/generator.rs` `collect_labels`

### 空格键在属性面板等控件输入时异常唤出搜索

**症状**：在右侧属性面板文本框中按空格时，搜索窗口被打开。Space 快捷键应仅在画布焦点或无文本输入时触发。
**文件**：`src/app.rs:update` 快捷键处理

---

## P3 — 视觉 / 优化

### 折叠后节点高度未缩小

**症状**：折叠状态下仅参数预览隐藏，节点 body 高度不变。建议折叠时节点自适应最小高度（标题栏 + 端口）。归入节点大小可调整计划。

### 导出 JSON 与"保存"路径混淆

**症状**："导出 JSON" 按钮写入新文件后，再按"保存 (Ctrl+S)" 仍保存到旧 `current_file`。用户期望"导出"行为类似"另存为"。
**建议**："导出 JSON" 成功后更新 `current_file`。
