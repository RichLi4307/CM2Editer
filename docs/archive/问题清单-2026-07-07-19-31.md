## P0：交互阻塞（功能不完整，影响主要 workflow）

### P0.1 框选视觉与风格错误 ✅ 已解决

**症状**：

- 空白处左键拖拽框选时，画布不显示选择框矩形。
- Shift/Ctrl + 空白处左键拖拽当前为 cross 风格（相交即选中），但需求为 window 风格（完全包含才选中）。
- 在节点上按住 Shift/Ctrl 再拖拽，仍然进入单节点拖动，而不是以该节点为起点开始框选。

**解决方式**：

- `src/ui/interaction.rs` 中 `BoxSelecting` 状态增加 `mode` 字段，按拖拽方向决定：左→右为 `Window`（严格包含），右→左为 `Crossing`（相交）。
- 拖拽过程中绘制半透明选择框（Window=蓝色，Crossing=绿色）。
- 节点上按住 Shift/Ctrl 拖拽时进入框选模式，不触发单节点拖动。
- Shift 为追加、Ctrl 为减选、无修饰键为替换选集。
- 第二轮：Crossing 模式改为虚线绘制。

---

### P0.2 多选节点无法整体拖拽 ✅ 已解决

**症状**：框选多个节点后，只能拖动鼠标按住的那个节点，其他被选节点不会跟随移动。

**解决方式**：`DraggingNode` 状态记录所有被选中节点的起始位置，拖拽时同步偏移；释放后为每个实际移动的节点生成 `MoveNode` 命令。第二轮：修复图层顺序，选中节点置顶渲染。

---

### P0.3 连线无法选中/删除 ✅ 已解决

**症状**：

- 鼠标悬停或单击连线无法将其选中。
- 框选（cross/window）无法选中连线。
- 选中连线后按 Delete 无反应。

**解决方式**：

- `EdgeRenderer` 新增 `hit_rect` 方法，对贝塞尔曲线采样并返回外接命中框。
- `app.rs` 渲染边时收集 `edge_hits` 并传入 `handle_input`。
- 单击连线选中、框选包含连线、选中后按 Delete 删除，`delete_selected` 已支持删除边。
- 第二轮：连线改为在节点下层渲染，高亮用蓝色发光方案（底层辉光 + SELECTED_GLOW 核心色）。

---

### P0.4 端口高亮/吸附与不兼容提示缺失 ✅ 已解决

**症状**：

- 从输出端口拖线时，兼容输入端口不会高亮，也没有吸附效果。
- 拖到不兼容端口时不会变红，也没有视觉拒绝提示，仅松手后状态栏提示"目标端口不兼容或无效"。

**解决方式**：

- `DrawingEdge` 状态实时检测鼠标下端口，按兼容/不兼容/已占用/成环返回 `PortTargetStatus`。
- 兼容端口显示高亮圆环；不兼容/成环/已占用显示对应颜色（红/橙）。
- 临时拖线颜色随目标状态变化。
- 释放时拒绝非法连接（自环、成环、类型不兼容、Data 端口已占用）。
- 第二轮：拖线悬停时目标端口圆点填充状态颜色。

## P1：右键菜单与节点操作

### P1.1 右键菜单复制/粘贴未落地 ✅ 已解决

**症状**：

- 右键节点菜单中的"复制"仅选中该节点，并未复制到剪贴板。
- 复制后没有"粘贴"选项。
- Ctrl+V 无法粘贴节点。

**解决方式**：

- 新增 `Clipboard { nodes, edges }` 结构，`App` 持有该字段。
- "复制"将当前选区节点及节点间连线写入剪贴板，并重新生成 ID。
- "粘贴"在右键菜单位置或鼠标悬停处创建副本，节点间连线同步重建。
- 工具栏与全局快捷键支持 Ctrl+C / Ctrl+V。
- 第二轮：快捷键改用 `consume_key` 防止被文本输入消费；新增空白处右键粘贴菜单、Ctrl+S 保存快捷键。

---

### P1.2 右键删除节点未级联删除边 ✅ 已解决

**症状**：右键节点 → 删除，仅删除该节点，其关联边仍残留在图中和 JSON 里。

**解决方式**：右键菜单"删除"收集该节点的所有关联边，以 `Command::RemoveNode { node, edges }` 入栈，撤销时可恢复节点及其边。

## P2：文件与数据一致性

### P2.1 加载 JSON / 导出 `.code` 未接入 GUI ✅ 已解决

**症状**：

- 顶部工具栏只有"保存 (Ctrl+S)""导出 JSON""导出 .code"按钮。
- 没有"打开/加载 .json"按钮或菜单。

**解决方式**：

- 工具栏新增"打开 JSON"按钮，使用 `rfd` 弹出文件选择对话框，加载后替换当前图并恢复视口。
- 保存 JSON 的 `version` 由 `serializer::migration::CURRENT_VERSION` 控制（当前为 `"1.3"`）。
- 导出 `.code` 与 `docs/examples/` 的对照由现有 `tests/code_gen.rs` 覆盖。
- 第二轮：保存 JSON、导出 JSON、导出 .code 均改为 `rfd` 文件保存对话框。

- 工具栏新增"新建"按钮可清空当前图。

## P3：极端场景与验证器可视化

### P3.1 环检测无画布提示 ✅ 已解决

**症状**：故意连接成环（A→B→A）时，底部状态栏显示错误文字，但画布区域无任何高亮或提示。

**解决方式**：

- `GraphValidator::check_no_cycles` 返回环上节点 ID 列表，`FlowError::CycleDetected(Vec<String>)`。
- `App::validate` 提取受影响节点 ID 到 `error_nodes`。
- 错误节点渲染红色边框；状态栏显示错误数量。

---

### P3.2 复制粘贴性能未验证 ⚠️ 需重新测试（已加视口裁剪）

**症状**：100 个 Start 节点后帧率低于 20（无论是否在屏幕内）。

**说明**：第二轮新增视口裁剪（仅渲染画布边距 50px 内节点和连线），需要重新测试。

---

### P3.3 损坏 JSON 加载未处理 ✅ 已解决

**症状**：加载损坏或版本不支持的 JSON 时，程序可能 panic 或无任何友好提示。

**解决方式**：`load_json` 捕获读取/解析/迁移错误，通过 `status_message` 显示"加载失败: ..."，程序不 panic。
